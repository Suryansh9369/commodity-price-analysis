# 📊 Commodity Market Intelligence System

## 🔷 Project Overview

This project builds a data-driven system to analyze and understand commodity price behavior for **gold, silver, and crude oil**.
It focuses on identifying **trends, volatility, and relationships** between assets to support **risk-aware investment decisions**.

---

## 🎯 Problem Statement

Commodity prices are highly volatile and influenced by multiple factors.
Investors and analysts often struggle to:

* Understand long-term trends
* Compare risk across assets
* Identify relationships between commodities

This project addresses these challenges using historical data analysis.

---

## 🚀 Objectives

* Analyze price trends of gold, silver, and oil
* Measure volatility (risk) of each commodity
* Identify correlations between assets
* Compare performance across different time horizons
* Build a foundation for forecasting models

---

## 🛠️ Tech Stack

* **Python**
* **Pandas, NumPy** (Data Processing)
* **Matplotlib, Seaborn** (Visualization)
* **SQLAlchemy + SQLite** (Database)
* **yfinance API** (Data Collection)
* **Git & GitHub** (Version Control)

---

## 🔄 Project Workflow

1. **Data Collection**

   * Fetch historical data using yfinance API

2. **Data Pipeline**

   * Logging system for tracking ingestion
   * Store data directly into SQLite database

3. **Data Processing**

   * Clean and structure time-series data
   * Set Date as index

4. **Exploratory Data Analysis (EDA)**

   * Trend analysis
   * Distribution (KDE, histograms)
   * Volatility analysis
   * Correlation between commodities

---

## 📈 Key Analysis

* Trend visualization of commodity prices
* Scatter plots with regression and density
* Distribution analysis using KDE
* Correlation matrix between gold, silver, and oil

---

## 📊 Insights (To Be Expanded)

* Gold tends to be more stable compared to oil
* Silver shows correlation with gold
* Oil exhibits higher volatility
